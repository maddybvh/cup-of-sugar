# cup-of-sugar

## Installation

- `git clone git@github.com:maddybvh/cup-of-sugar.git`
- `cd cup-of-sugar`
- Obtain `secrets` add add them to src/constants/secrets.js
- `yarn`
- `yarn start`
- visit http://localhost:3000

## Deployment

- Install [Netlify CLI tools](https://docs.netlify.com/cli/get-started)
- `yarn build`
- `netlify deploy`
